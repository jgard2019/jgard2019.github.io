Virtual Pet Application

The pet app exercise was a group project most likely chosen because it requires a firm grip on Java basics. We were tasks with making an in-console game that prompts the player to care for virtual pets by tending to the pets basic needs. Introduced early on in the bootcamp, we broke off into small groups after what felt like a few crash courses in Java and minimal instruction. "Have at it!" they said.

I'm paraphrasing.

Our successful completion of the project utilized (to name a few) while loops (for the primary game loop), ArrayLists (to store objects, such as the game's pets), constructors, getters, and interfaces. It also had a few dozen methods to manipulate object fields and add a dynamic sense of activity within the game.
