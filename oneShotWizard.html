DND Character Creater App

Return To Home

Project Details

Full Stack Application. This group project was a triumph in so many ways. The plan was to create a single-page application that creates a character sheet for Dungeons and Dragons (the table top game). Basically the app allows you to select many specific properties for your character, and save them all to a database to be viewed or referenced as needed. The application saved all "characters" created and lets users browse through them. The site also leveraged a third party API that populates a tremendous amount of information about the game to be reviewed at any time.

Project hosted on Heroku
