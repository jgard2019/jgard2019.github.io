Virtual Pet Project

Return To Home

Project Details

A simple console application heavily based on the "Tomagatchi" digital pets that were so popular in the 90's. The project's goal was to teach new programmers how to implement many of the basic principles of programming. Using only Java and an IDE and the console, I was tasked with coding a Virtual pets shelter where the player had to feed, water, and entertain pets. The player could add or subtract many types of pets to the shelter, and any actions taken by the user would simulate the passage of time which would increment the needs of the sheltered pets.

Languages Used

Java, in Intellij Idea

Programmng Principles Applied

  • Encapsulation (Each pet was its own Class)
  • Inheritence
  • Abstraction (Generic classes would inherit fields and methods from super class)
  • Interfaces / Polymorphism
