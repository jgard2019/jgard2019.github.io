Website Plan

Website Topic and Title

The Title of this website project will be "Digital Retreat"

Target audience is as follows:

  • Primarily Myself
  • Close friends
  • Some Family
  • Anyone who happens to stumble upon my random writing and articles

Site Purpose

The purpose of this site is to serve as a repository for my thoughts, opinions, and remembrance. It is mainly meant to serve as a personal digital journal.

Comparable Sites

  • www.blogger.com
  • www.livejournal.com
  • www.tumblr.com

Main Site Content

The majority of the content of this site will include:

  • Mostly writing, journaling, prose, ramblings
  • Photographs I consider to be significant, along with the stories behind them
  • Writing about opinions and reviews on things like cars and technology

Site Personality

Obviously a website like this will have a lot of personality. It will eventually serve as a showcase for my web-development skill and styles. I personally can't imagine how anyone learns how to code websites or any computer software without having some level of passion for this type of work. I've mentioned it before in one of my earlier web-projects in this class: There won't always be total freedom of design and creation when this becomes a profession. But there's nothing stopping me from staying up late one night, or getting up early one morning to sit and create something. This site should serve as my own personal creative space. And at the points where I don't have time to upkeep this as much as I'd like, it will serve as a reference to how I used to write code and design websites. So the website personality will be catered to:

  • Highly legible main content
  • Minimal Navigation
  • Emphasis on beautiful web design, but also minimalistic or at least not gaudy
  • Attention to little subtle details, like someone, you know, cares about it

Limitations

Designing with Mobile in mind, always. I wouldn't create a site that I personally can't access anywhere. So there should not be issues with loading this website on older browsers and slower internet connections.
